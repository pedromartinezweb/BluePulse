# BluePulse user guide

## Download

Download `BluePulse.exe` from the latest GitHub release:

https://github.com/pedromartinezweb/BluePulse/releases

Until SignPath approval is complete, release builds are published unsigned.

## Run

1. Open `BluePulse.exe`.
2. Enable the utility.
3. Choose the idle threshold in minutes.
4. Minimize BluePulse to the system tray if you want it out of the way.

No installer is required.

## System tray

When minimized, BluePulse stays available from the Windows system tray. Use the tray menu to open, enable, disable, or exit the app.

## Safety model

BluePulse does not:

- Simulate keyboard input.
- Simulate mouse input.
- Install a background service.
- Change Windows policy settings.
- Bypass corporate lock policies.
- Collect user data.
- Send network requests.

If a company policy forces idle locking, BluePulse does not disable or bypass that policy.
